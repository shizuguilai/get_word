# 中文跟读提词器

一个零依赖的手机端 PWA 提词器，只启用中文语音识别。粘贴稿件后进入提词模式，按“开始跟读”，应用会根据识别到的中文自动高亮并滚动到当前朗读段落。

## 运行

在本目录启动本地服务：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4173
```

桌面本机测试可以直接使用这个地址。手机真机使用麦克风时，建议部署到 HTTPS 静态站点后打开，再用浏览器菜单添加到主屏幕；普通局域网 `http://电脑IP:4173` 可能会被手机浏览器拦截麦克风权限。

## 语音识别说明

当前版本使用手机浏览器或系统提供的中文语音识别能力，语言固定为 `zh-CN`。Chrome / Edge Android 通常可用；如果浏览器不支持 `SpeechRecognition`，应用会提示更换浏览器。

大型中文离线 ASR 模型通常有几十 MB 到数百 MB，直接塞进前端会让手机首次打开很慢。项目已把识别逻辑集中在 `app.js` 的识别引擎部分，后续可以替换为 Vosk、sherpa-onnx 或其他中文本地模型适配器。

如果要改成真正下载离线模型，建议选中文专用的小模型，并把模型初始化后的实时/分段识别结果送进 `advanceBySpeech(text, isFinal)`。这能复用现有的中文规范化、模糊匹配、自动高亮和滚动逻辑。
